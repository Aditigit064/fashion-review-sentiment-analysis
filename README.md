# Fashion Review Sentiment Analysis

## Overview
Built an NLP-based sentiment analysis model on 23K+ fashion product reviews to classify customer feedback into positive, neutral, and negative sentiment.

## Tools Used
- Python
- Pandas
- NLP
- Scikit-learn
- Jupyter Notebook

## Model Used
- TF-IDF Vectorization
- Logistic Regression

## Key Insights
- Most reviews were positive
- Common complaints involved sizing, fabric, and comfort
- Negative sentiment was concentrated in specific product categories
